## Instructions
1. **Prerequisites**
   - Ensure that you have Node.js installed on your computer. You can download it from [here] (https://nodejs.org/es/).

## How to Use

   ```bash
   # Clone this repository
   $ git clone https://github.com/peterleunam1/viio-test
   # Install dependencies
   $ npm install 

   # Run app
   $ npm run dev 

   # Run tests
   $ npm run test

   # Build app
   $ npm run build
   $ npm run start
   ```
